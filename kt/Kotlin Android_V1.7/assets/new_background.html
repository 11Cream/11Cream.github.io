Background:


Kotlin is designed by JetBrains and developed by JetBrains and open source contributions.

Kotlin first appeared in the year 2011, but got its first stable release in the year 2016 i.e. version v1.0.


Current version of kotlin: 1.1.2


Kotlin works on JVM(Java Virtual Machine ), bytecode and JavaScript source code platforms.


About JetBrains:

JetBrains(Formally IntelliJ) is a software development company.

JetBrains have given us the product like IntelliJ, Android Studio, Appcode, Clion and now Kotlin.

Tools

IntelliJ IDEA has plug-in support for the kotlin. IntelliJ IDEA 15 is the first version to bundle Kotlin plugin in the b>IntelliJ Installer and provide support of it out of the box.

JetBrains also provides a plugin support forEclipse


Kotlin became the third language fully supported for Android, after Java and C++.
